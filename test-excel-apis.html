Excel Import/Export API Test Page

1. Export Materials to Excel

Endpoint: GET /api/materials/export
Description: Export materials filtered by date range

2. Import Materials from Excel

Endpoint: POST /api/materials/import
Description: Import materials from an Excel file
Required columns: Material (B), Material Description (C), Available Stock (E)

3. Export Parts to Excel

Endpoint: GET /api/parts/export
Description: Export parts filtered by date range

4. Import Parts from Excel

Endpoint: POST /api/parts/import
Description: Import parts from an Excel file
Required columns: Lear PN (B), Tesca PN (C), Description (D), Qty Per Box (E)

5. Export Combined Tickets and Ticket Codes

Endpoint: GET /api/tickets-combined/export
Description: Export tickets combined with ticket codes (joined on ticket.ticketCode = ticketCode.code)
